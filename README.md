# Inspire Claude Marketplace

Plugin marketplace for Inspire Software clients.

## Installing

Add this marketplace in Claude Code or Cowork:

```
/plugin marketplace add Inspire-Software-Solutions/claude-marketplace
```

Then install the Inspire plugin:

```
/plugin install inspire@inspire-plugins
```

## Plugins

| Plugin | Description |
|--------|-------------|
| `inspire` | Connect Claude to your Inspire account — read data and create records directly from your conversation |

## Updates

To get the latest version of any plugin:

```
/plugin marketplace update inspire-plugins
```
